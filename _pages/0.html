Design system

Halton's Design System is created to provide a stable and detailed documentation, to symplify the usage of custom components, and to facilitate their implementation by any content creator.

Scroll over the component to copy the code.

Buttons

Custom buttons created for the Halton brand. Icons can be changed at will by using any icon classes listed below. Icon can be moved left and right by using the class icon-before or icon-after. The primary button is acheived by adding the class halton-btn-primary
